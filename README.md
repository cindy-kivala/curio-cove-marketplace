# CurioCove Marketplace

A collectible trading platform where buyers and sellers can negotiate prices and chat in real-time.

## Features
- Browse collectible items
- Make offers on items
- Real-time chat between buyers and sellers
- Seller dashboard to manage listings
- Payment confirmation flow

## Tech Stack
- Backend: Node.js + Express
- Frontend: Lit Web Components
- Styling: Tailwind CSS
- Templating: EJS

## Setup
1. `npm install`
2. `npm run dev`
3. Visit `http://localhost:3000`
